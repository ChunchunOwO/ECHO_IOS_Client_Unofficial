# Release 更新日志

## 0.4.0 - 音乐播放器重构、设置页与专业音频 Tag

发布日期：2026-06-23

这一版把 ECHO iPhone 从“遥控器式界面”继续推进成更完整的音乐播放器伴侣，重点处理播放页拥挤、组件遮挡、专业音频信息展示和基础设置入口。

### 新增

- 新增设置页，并加入底部 dock。
- 设置页支持界面语言选择：中文 / English。
- 设置页支持选择音频 tag 展示项：输出模式、来源、串流能力、格式音质、码率、时长。
- 曲库新增筛选：全部 / 可串流 / 本地。
- 连接页新增 Host、曲库数量、可串流数量概览。

### 音频信息

- 支持展示 ECHO 桌面端发送的专业音频字段：`codec`、`sampleRate`、`bitDepth`、`bitrate`。
- 音频质量 tag 合并为更易读的格式，例如 `FLAC 48kHz/24bit`。
- 兼容 `48000` 和 `48` 两种采样率输入，避免显示异常。

### 播放体验

- 播放页整体重构为深色玻璃音乐播放器风格。
- 音量控制改为辅助控制行里的可展开滑条，减少常驻占位。
- 上一首 / 下一首改成双箭头，降低与页面滑动或返回手势的视觉冲突。
- 歌词页保留大字号、高对比、自动滚动与点击歌词跳转。
- 播放列表弹窗保持在播放页上方展示，并修复暗色背景下标题对比。

### 布局与稳定性

- 收紧封面、主控按钮、tag 与控制区尺寸，降低小屏重叠风险。
- 歌词页或音量展开时允许滚动，避免 dock 遮挡控制区域。
- 连接页 Host 文本增加单行截断，避免撑破布局。
- 保持手机端改动独立，没有修改 PC 端源码。

### 验证

- `npm.cmd run typecheck`
- `git diff --check`
- `npx.cmd expo export --platform ios --output-dir build\export-check`

### 已知限制

- 语言设置目前先覆盖导航、页面标题/描述和设置页文案，播放页内部分功能文案仍保持中文。
- 设置项暂未持久化，重启 App 后会恢复默认显示。

## 0.3.0 - 全局 UI 重构与文档刷新

这一版把 ECHO iPhone 的视觉和文档都重新整理了一遍，让它更像一个完整的音乐播放器伴侣，而不是原型页面。

### 新增

- 播放页、歌词页、dock 栏统一重构为一套玻璃感界面。
- 播放页改为更满、更紧凑的满版布局。
- 歌词页重做为高对比阅读面板，字号更大，当前行自动滚动。
- 预览图改成更偏 ACG / 二次元气质的功能海报。
- 应用图标更新为更干净的深色音符风格。
- README 和英文 README 重新写成更完整的项目主页风格。

### 优化

- 歌曲封面加载更稳定，不再频繁闪回默认占位图。
- 进度条和音量条拖动时锁定页面手势，减少断触和误滑动。
- dock、按钮、歌词区和播放面板统一成更一致的视觉语言。
- 预览图改为直接介绍功能：EchoLink、歌词、曲库、串流、IPA 构建。

### 继续保留

- EchoLink 配对与手动连接。
- 曲库浏览、点歌、播放控制。
- 手机串流、单曲循环、播放列表预览。
- 未签名 IPA 的 GitHub Actions 构建流程。

### 验证

- `npm.cmd run typecheck`
